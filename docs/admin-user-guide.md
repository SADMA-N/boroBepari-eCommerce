# Admin User Guide

## Access & Roles
Roles are enforced via RBAC:
- Super Admin: full access
- Admin: most features, no settings edits
- Moderator: KYC review and product moderation only

## Core Workflows
- Users: review accounts, suspend/delete with confirmation
- Suppliers: verify KYC, manage status, review performance
- Products: moderate listings, handle flags, suspend/delete
- Disputes: resolve refunds, request info, escalate
- Audit Logs: review all admin actions

## Security
- 2FA is required for admin login
- Sessions timeout after inactivity
- Critical actions require confirmation

## Tips
- Use filters to narrow large queues
- Export CSV for compliance reviews
- Check audit logs for accountability
